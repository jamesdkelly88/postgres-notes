# Order of Execution

- FROM : specify source table
- JOIN : specify additional tables
- WHERE : filter rows
- GROUP BY : aggregate
- HAVING : filter groups
- SELECT : specify columns
- DISTINCT : remove duplicates
- ORDER BY : sort output
- LIMIT : reduce output rows
- OFFSET : skip rows
